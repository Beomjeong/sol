쿠폰 등록방법

1.

게임 접속 후 [메뉴] → [환경 설정] → [계정]에서 [쿠폰 등록] 버튼을 선택합니다.

2.

쿠폰 번호를 입력하거나 붙여넣어 쿠폰을 사용합니다.

※

게임 내 등록 시 별도의 PID 입력은 필요하지 않으며, 현재 접속된 캐릭터 기준으로 보상이 지급됩니다.

※

애플 정책에 따라 iOS(애플) 기기 사용자는 인게임에서 쿠폰 등록을 사용하실 수 없습니다.

1.

SOL: enchant 쿠폰 등록 페이지에 접속합니다.

2.

계정 정보(PID) 입력 후 [확인] → 보상 받을 서버 / 캐릭터를 선택합니다.

3.

쿠폰 번호 입력 후 [사용하기] → [확인] 을 클릭하면 등록이 완료됩니다.

※

회원 번호(PID)는 게임 내 [메뉴] → [환경 설정] → [계정]에서 확인 및 복사할 수 있습니다.

✓

쿠폰은 2026년 8월 22일까지 등록 가능하며, SOL: enchant 계정 당 1회 등록 가능합니다.

✓

쿠폰을 통해 획득한 보상은 선택한 서버(월드)·캐릭터의 우편함으로 지급됩니다.
